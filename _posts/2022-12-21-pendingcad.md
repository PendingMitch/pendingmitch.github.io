## What is PendingCAD?
PendingCAD is a project that will allow better roleplay features to civilians, police, medical and fire.

This project is only developed by myself and will continue as such. 

If you have any questions feel free to contact me and [#make-a-ticket](https://discord.com/channels/1004157797627334666/1004162225063329924)


## Why did this project start?

This project was started from inspiration from HamzCAD and SonoranCAD. One day, I was playing around on FivePDRP's server. The SonoranCAD that they used had gone down for maintenance. I wanted to offer a service that wouldn't go down away from its main server.

The closest option I saw to this was HamzCAD. But there was a few issues that I saw with that CAD. For starters, it was written in PHP which was a language that I didn't know at the time. Secondly, because of this I couldn't make as many changes to the CAD that I would've perhaps liked to.

This lead me to creating my own CAD in the technology I was recently learning named 'ReactJS'. To make it easier for myself with the API calls and backend I used 'NextJS'.

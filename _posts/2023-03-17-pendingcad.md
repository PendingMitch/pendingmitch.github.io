## PendingMedical

A simple Computer Aided Dispatch / Mobile Data Terminal system for your FiveM Needs.

### Inspirations

This project was inspired from SonoranCAD and HamzCAD.

The issue that I had with SonoranCAD is the inability to host the application on your own server. This meant that it was more likely to suffer from down times that were not convinent to you.

The issue that I had with HamzCAD is the language and licenses that surrounded it. I created this application in a language (javascript) that I knew and so I was able to change the features that I wished to have in the application.

### Try it out

Try it out at [PendingCad](https://fivem-cad-js.vercel.app/).

Please be aware that you will need to sign in with an account - this is to protect the service that PendingDevelopment provides.
